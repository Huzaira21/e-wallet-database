# E-Wallet with Database

A mini e-wallet system built in C++ and SQL with secure 
transactions and account management features.
Developed at Namal University, Mianwali.

## Features
- Secure user authentication
- Transaction management
- Account balance tracking
- Database integration with SQL
- Data persistence
